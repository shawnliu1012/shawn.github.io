介紹JS及瀏覽器運作

JS簡介

  • JS為瀏覽器唯一指定的內建程式語言
  • 與java毫無關聯，只是名字像!

    瀏覽器如何運作

    可使用開發者工具的network下方的All檢查，重新整理之後點選第一個觀察request&response。[color=#90ddff]

javascript 與 jQuery 的關係

  • jQuery 是一個JS函式庫(library)
  • jQuery提供許多方便的方法
  • 但JS也可以得到相同效果

    HTTP請求方法與HTTP動詞

  • GET:用於平時網頁瀏覽時輸出資料不受保護(對比post)的情形。
  • POST:用於輸入帳密時，受保護的狀態下需要傳送資料給伺服器驗證的情形。
  • DELETE:假如線上購物時，希望刪除商品時的情形
