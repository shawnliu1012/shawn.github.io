認識 HTML、CSS 與實作第一個網頁

介紹什麼是HTML及CSS，並且實作第一個網頁

什麼是HTML?

  • HTML為Hypertext Markup Language縮寫。
  • 用來組織架構並呈現網頁內容的標記式語言。
  • 為標記式語，非程式語言。
  • HTML碼是由<>內字元所組成的，稱為HTML元素(element)。
  • 標記通常式成對的。開始標記代表了內容開始，結束標記代表內容結束。(結束標記會多一個”/“)
  • 開始標記可以乘載屬性，以進一步去說明該元素的內容。

什麼是CSS?

  • CSS是一組網頁建構基礎樣式，CSS 主要增加了更多的樣式定義方式來輔助 HTML語言,CSS 僅是在既有的網頁HTML語言中，加上一些輔助語法(Plugins)，藉以達到控制網頁外觀的另一個簡便的方式。
  • 規則是用選取器(指定規則要套用到那些元素上)和宣告(指定這些元素的外觀)構成
