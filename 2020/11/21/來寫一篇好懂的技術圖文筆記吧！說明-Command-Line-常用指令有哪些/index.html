什麼是Command line及說明 Command Line 常用指令

什麼是command line

command line interface (命令列介面或命令行介面，簡稱 CLI) 使用純文字的指令來操作電腦的介面，來與電腦溝通，為非圖形化的介面。使用command line指令有助於在操作複雜程序時，更加快操作時間。

而graphical user interface(圖形使用者介面，可以簡稱 GUI)，則為一般看到電腦畫面的資料夾。

command line 常用指令有哪些

  • pwd
    查看自己的路經(Where am I)

    可使用 pwd --help查看詳細指令

  • ls
    可列出資料夾內的項目

    ls相關常用指令包含:
    -l(除了查看項目更包含查看擁有者、權限、建立時間等詳細資 料)
    -a(顯示所有隱藏的檔案)
    可使用ls --help查看詳細指令

  • cd (後接目錄/資料夾)
    更換當前的目錄

    cd相關常用指令包含:
    cd . .移動到上一個目錄
    cd . ./. .移動到上上目錄

  • rm
    刪除檔案(須加檔案類型)

  • rmdir
    刪除資料夾

    強制刪除(比較危險，慎用)，用於快速將資料夾(包含資料夾內)所有檔案刪除。

  • cp(file1 flie2)
    複製檔案
